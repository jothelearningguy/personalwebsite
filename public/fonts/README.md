# Neue Machina Font Files

To use Neue Machina on the blog page, please download the font files from:

**Pangram Pangram Foundry**: https://pangrampangram.com/products/neue-machina

## Required Font Files

Place the following font files in this directory:

- `NeueMachina-Regular.woff2`
- `NeueMachina-Regular.woff`
- `NeueMachina-Regular.otf`
- `NeueMachina-Bold.woff2`
- `NeueMachina-Bold.woff`
- `NeueMachina-Bold.otf`
- `NeueMachina-Light.woff2`
- `NeueMachina-Light.woff`
- `NeueMachina-Light.otf`

## License

Neue Machina is free for personal use. For commercial applications, a license must be purchased from Pangram Pangram.

## Alternative: Using a CDN

If you prefer to use a CDN instead, you can modify the `@font-face` declarations in `src/components/Blog.css` to point to a CDN URL.
